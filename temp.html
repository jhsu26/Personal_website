    • Technologies used: HTML, CSS, Javascript
    • Description: I completed this programming challenge for the internship program at KL Software Technologies.   • Greenwood Genetic Center - Web Development Intern

    • Duration: June 2015 - August 2015
    • Location: Greenwood, SC
    • Description: I worked under the supervision of Dr. Chin-fu Chen (Center for Molecular Studies and Office of Bioinformatics and Epidemiology, Greenwood Genetic Center, Greenwood, SC, 29646) and Dr. Gilliean Lee (Department of Mathematics & Computing, Lander University, Greenwood, SC, 29649) on MICO: Mutation Information Collector, an online tool that collects mutation information. At the end, I gave a presentation about my work.
    • Project details:
      • MICO single mutation: implementing new and responsive user interface, writing new Java classes for sending queries to five new bioinformatic tools, and implementing the five new tools on the servlet.
      • MICO batch query: Developing new and responsive user interface, writing new Java classes for sending queries to three bioinformatic tools with batch queries, implementing the three tools, and developing the servlet used.
      • Using JavaFX to add three new bioinformatic tools to MICO mini, the desktop version of MICO.
      • Writing three programs that: converted a .txt file full of FASTA DNA & protein sequences into .csv files that are importable to MySQL, takes in a protein ID (i.e. Ensembl Peptide ID) and retrieves its FASTA protein sequence, and takes in a protein ID input and converts it to another type of protein ID (i.e. Ensembl peptide ID to RefSeq peptide ID).
      • Updating the MySQL database for MICO by creating two new tables and updating one existing table.
